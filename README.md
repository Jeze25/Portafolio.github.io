## 🚀 Cómo Ejecutar Localmente

1. Clona el repositorio:
```bash
git clone https://github.com/Jeze25/Portafolio.github.io.git
```

2. Navega al directorio:
```bash
cd Portafolio.github.io
```

3. Abre el archivo `index.html` en tu navegador o usa un servidor local:
```bash
python -m http.server 8000
# o con Node.js
npx http-server
```

4. Abre `http://localhost:8000` en tu navegador

## 📝 Personalización

Para personalizar el portfolio:

1. **Editar información personal** en `index.html`
2. **Cambiar colores** en la sección `:root` de `styles.css`
3. **Agregar nuevos proyectos** copiando las tarjetas de proyectos existentes
4. **Actualizar enlaces** de redes sociales y contacto

## 🎨 Personalización de Colores

Las variables de color están definidas en `styles.css`:

```css
:root {
    --primary: #6366f1;           /* Color principal (Indigo)*/
    --primary-dark: #4f46e5;      /* Color primario oscuro */
    --secondary: #ec4899;         /* Color secundario (Rose) */
    --background: #0f172a;        /* Fondo principal */
    --surface: #1e293b;           /* Fondo secundario */
    /* ... más variables */
}
```
